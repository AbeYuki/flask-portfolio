Infrastructure

オンプレミスで商用環境・検証環境、Oracle Cloud でステージング環境、さくらクラウドで検証環境を構築しています。
インフラ構築について、オンプレミスは Ansible でサーバー周りの管理を行い、Oracle Cloud については Terraform でクラウドの展開を行っています。
アプリケーション展開はどの環境も Terraform helm provider で Argo CD のデプロイを行い、Argo CD でアプリケーションの展開を行っています。
